C++ Number System Converter

Link To Project

  I had a goal in mind over summer 2021, that was to learn new programming languages. This is my number system converter that I made myself in C++ for the purpose of learning C++ and making something useful. The program prompts the user to select either binary, decimal, hexadecimal, or octal, and for the user to input a number in their chosen number system. After the user inputs their number, they can convert it to any of the other number systems provided. This project was quite challenging and enjoyable to create, helping me develop a better understanding of C++ but also problem solving in general.
