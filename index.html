Snap! Project

Link To Project

  This was a project I worked on in UC Berkeley’s “Snap!”. My goal was to parlay the basic “Snap!” library into something more comprehendible, I accomplished that goal by making this video game. The game is simple, you control a basket that can either move left or right attempting to catch as many falling apples as possible. In making the simple game, I practiced efficient coding methods, comprehended the “Snap!” library, and created a better understanding of coding for myself. And while it is quite simple, its basic structure highlights the simplicity yet endless possibilities of coding.
